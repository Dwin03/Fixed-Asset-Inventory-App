# Fixed Asset Inventory System 📱

A system designed to manage and track fixed assets such as equipment, tools, and company property. Developed as an practical exam project, it demonstrates database handling by combining CSV file processing with Room Database for efficient data management, editing, and storage.

---

## 🚀 Features

* Add and manage fixed asset records
* Store asset data using Room Database
* Import data using CSV file
* Export data in CSV file format
* Update and delete asset information
* View complete list of assets

---

## 🧠 How It Works

1. User inputs asset details (name, category, value, etc.).
2. The system stores data in a Room Database.
3. Asset records can also be exported or handled via CSV files for external use.

---

## 🛠️ Tech Stack

* Kotlin - Backend
* Room Database - Local database
* CSV Files – Data umport and export
* Android Studio - Coding Platform

---

* CSV handling may not support large datasets efficiently
